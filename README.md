### 프로젝트 설명
<div align="center">
  <img src="https://github.com/sangzun-han/apple-store/assets/63501931/b92ec405-6c35-4dec-b044-57917b23d3a6" width="400" />

1. TodoMain
  - Todo 리스트의 레이아웃 설정
2. TodoHead
  - 오늘의 날짜와 요일 보여주기
3. TodoList
  - 할 일에 대한 정보가 들어 있는 todos 배열 보여주기
4. TodoItem
  - 각 할일에 대한 정보를 렌더링 해주는 컴포넌트
5. TodoCreate
  - 새로운 할 일을 등록할 수 있는 컴포넌트
